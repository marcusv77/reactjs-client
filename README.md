## How to run
```bash
# Clone Repository
$ git clone https://github.com/marcusv77/reactjs-client.git
```
### Run Web Project

```bash
# Go to web folder
$ cd reactjs-client

# Install Dependencies
$ yarn install or npm install

# Run Aplication
$ yarn start or npm start
```
Go to http://localhost:3000/ to see the result.
