## How to run
```bash
# Clone Repository
$ git clone https://github.com/marcusv77/api-client.git
```
### Run API

```bash
# Go to server folder
$ cd api-rest/api

# Install Dependencies
$ yarn install or npm install

# Run Aplication
$ yarn start or npm start
```
Access API at http://localhost:3333/

### Run Web Project

```bash
# Go to web folder
$ cd api-rest/client

# Install Dependencies
$ yarn install or npm install

# Run Aplication
$ yarn start or npm start
```
Go to http://localhost:3000/ to see the result.
